# Asset Tracking Web Application

Prototype web app for indoor asset tracking using BLE tags.

## Tech Stack
- **Frontend:** React, TailwindCSS, React-Konva
- **Backend:** Node.js + Express + WebSocket



## Quick Start

### 1. Clone Repo

```bash
git clone https://github.com/<your-username>/AssetTracking.git
cd AssetTracking
```

### 2. Setup Backend
```bash
cd backend
npm install
npm run dev    
```


### 3. Setup Frontend
```bash
cd ../frontend
npm install
npm start

```
